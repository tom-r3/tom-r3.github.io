Recreation CRM is a Customer Relationship Management web app co-written with 3 other developers.

It is written to integrate with a suite of other applications that together manage all of the businesses functional requirements. There is a portal available that provides a graphical overview of the company as a whole and individualized metrics and suggestions based on the RFM model for each customer. There is also an API available at /docs that can be used to add customers, purchases, promotions, events, customer interactions and customer information to our database as well as request analytics information.

The app is written in Python using the Django Web Framework and is hosted through and run inside a Heroku dyno.
