A single-cycle MIPS Processor.

It implements a number of MIPS assembly instructions such as:

  • load word
  • store word
  • move
  • add immediate unsigned
  • add unsigned
  • subtract unsigned
  • load immediate
  • jump
  • jump and link
  • jump register
  • set on less than immediate
  • branch equal to zero
  • branch no equal to zero
  • shift left logical

Plans are to extend this project in the future to create a fully pipelined processor and an ALU unit.

The processor is modelled in System Verilog.
